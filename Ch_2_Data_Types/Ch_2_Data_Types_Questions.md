JavaScript Practice – Data Types & Type System
==============================================

Level 1 – Basic Understanding
-----------------------------

### Q1. What are data types in JavaScript?

**Answer**

Data types define the kind of value a variable holds, such as number, string, boolean, object, etc.

There are two main categories:

*   Primitive types
    
*   Reference types
    

### Q2. What are primitive data types?

**Answer**

Primitive data types are stored directly in memory.

Types:

*   String
    
*   Number
    
*   Boolean
    
*   Undefined
    
*   Null
    
*   Symbol
    
*   BigInt

### Q3. What are reference data types?

**Answer**

Reference types store values as memory references instead of actual values.

Types:

*   Object
    
*   Array
    
*   Function

Common Confusions (Important)
-----------------------------

*   typeof null === "object" → bug
    
*   "5" + 1 = "51" but "5" - 1 = 4
    
*   undefined ≠ null
    
*   NaN is of type "number"
