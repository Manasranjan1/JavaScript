


---

### 🟢 Easy: Basic Syntax
- [ ] **1. The Count Up:** Write a `for` loop that prints numbers from 1 to 10.
- [ ] **2. The Even Split:** Write a `while` loop that prints only even numbers between 1 and 20.
- [ ] **3. Array Walker:** Create an array of 5 fruits and print each one using a `for-of` loop.
- [ ] **4. Name Barker:** Use a `for-of` loop to print every character of the string `"Sheryians"` in uppercase.

---

### 🟡 Intermediate: Logic & Control
- [ ] **5. The Skip:** Write a loop from 1 to 10, but use `continue` to skip the number 5.
- [ ] **6. The Exit:** Write a loop from 1 to 100, but use `break` to stop the loop as soon as you hit 22.
- [ ] **7. Object Inspector:** Create a `user` object with `name`, `age`, and `role`. Use a `for-in` loop to print the keys and their values (e.g., `"name : Harsh"`).
- [ ] **8. Total Sum:** Create an array of numbers. Use `forEach` to calculate the total sum of all elements and print it.

---

### 🟠 Advanced: Problem Solving
- [ ] **9. String Reverser:** Write a function that takes a string (e.g., `"apple"`) and uses a loop to return it reversed (`"elppa"`).
- [ ] **10. The Accumulator:** Use `forEach` to look at an array of numbers and create a *new* array that only contains the numbers greater than 50.
- [ ] **11. Pattern Builder:** Use nested loops to print a right-angled triangle of stars:
  ```text
  *
  **
  ***
  ****
  ```

---

### 🔴 Expert: Real-World Scenarios
- [ ] **12. The Guessing Game:** Use a `while` loop and `prompt()` (or a hardcoded variable) to simulate a game where the loop keeps running until the "User" guesses a secret number (e.g., 7).
- [ ] **13. Data Cleaner:**
  Given an array `[10, "20", 30, "40", 50]`, use a loop and `typeof` to print only the items that are actual **numbers**, skipping the strings.
