

### 🟢 Easy: Array Basics
- [ ] **1. The Creator:** Create an array called `fruits` with 5 different fruit names. Print the 3rd fruit in the list.
- [ ] **2. Quick Update:** Given `let colors = ["Red", "Green", "Blue"]`, change "Green" to "Yellow" using its index.
- [ ] **3. Push & Pop:** Start with an empty array `tasks`. Add "Coding", "Cleaning", and "Cooking" to it using `.push()`. Then, remove the last item using `.pop()`.
- [ ] **4. Start & Finish:** Use `.unshift()` to add "Exercise" to the beginning of your `tasks` array and `.shift()` to remove it.

### 🟡 Intermediate: Transformation & Search
- [ ] **5. The GST Mapper:** Given an array `prices = [100, 500, 700, 1000]`, create a new array `pricesWithGST` where every price is increased by 18% using `.map()`.
- [ ] **6. The Filter Club:** Given `let ages = [12, 18, 25, 8, 30, 15]`, use `.filter()` to create an array of ages that are 18 or older.
- [ ] **7. Find the One:** Create an array of objects representing students: `[{name: "A", marks: 45}, {name: "B", marks: 85}]`. Use `.find()` to get the student who scored more than 80.
- [ ] **8. The Existence Check:** Use `.some()` to check if any number in `[10, 20, 30, 42]` is greater than 40. Then use `.every()` to check if all numbers are positive.

### 🟠 Advanced: Reduction & Destructuring
- [ ] **9. Total Payroll:** Use `.reduce()` to calculate the total sum of this salary array: `[40000, 50000, 35000, 70000]`.
- [ ] **10. Array Destructuring:** Destructure the first two elements and "the rest" of the elements into a variable called `others` from this array: `["Apple", "Mango", "Banana", "Grapes", "Orange"]`.
- [ ] **11. The Spread Copy:** Create a copy of an array `original = [1, 2, 3]` into `duplicate` using the spread operator. Prove they are different arrays by pushing a value into `duplicate` and checking `original`.

### 🔴 Expert: Complex Logic
- [ ] **12. The Proper Sort:** Sort the following array numerically in ascending order: `[40, 100, 1, 5, 25, 10]`. Explain why `.sort()` without a function would fail here.
- [ ] **13. Data Cleaning:** Given `[1, null, 2, undefined, 3, " ", 4]`, use `.filter()` to remove all "falsy" or null/undefined values.
- [ ] **14. Unique Values:** (Challenge) Given an array with duplicates `[1, 2, 2, 3, 4, 4, 5]`, use a loop or a modern JS method to return an array with only unique values.
